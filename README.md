# 👑 Royal Atelier

Royal Atelier is a modern frontend web application built for a cosplay business that specializes in **custom wig commissions** and **full costume rentals**.  
The platform focuses on showcasing high-quality work while providing a smooth and intuitive inquiry experience for potential clients.

---

## ✨ Features

- 🎭 Wig commission showcase
- 👗 Costume rental listings
- 📸 Portfolio gallery of past works
- 📩 Inquiry form for client requests
- 📱 Fully responsive design

---

## 🧠 Purpose

This project was built as a **real client-based website**, with a focus on:

- Clean and modern UI/UX
- Responsive design across devices
- Component-based architecture using React
- Practicing TypeScript for scalability and maintainability

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
